# Clash Royale Card Analysis

## 📌 Objective
Analyze card attributes such as damage, cost, and special mechanics to derive gameplay insights.

## 📊 Dataset
- Source: Clash Royale Wiki
- 70 cards with multiple attributes

## 🔧 Data Cleaning
- Handled missing values using domain knowledge
- Extracted features (card level, spawn level, shield HP)
- Converted mixed formats (ranges, fractions)

## 📈 Key Insights
- Most cards do not spawn troops
- Higher cost cards tend to have higher damage
- Shield cards show better survivability

## 🛠 Tools Used
- Python
- Pandas
- Matplotlib / Seaborn
